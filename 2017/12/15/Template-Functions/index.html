Template Functions

模板函數

模板函數是一個很神奇的東西，它的作用是可以使一個函數成爲運算的模板，這樣的話，這個函數就可以支持各種類型的操作。

就像這個程序·······

#include<iostream>
int n, m;
double p, q;
template <typename T>
T Bing(T, T);
int main() {
    std::cin >> n >> m;
    std::cout << "int類型的值:  " << Bing(n, m) << std::endl;
    std::cin >> p >> q;
    std::cout << "double類型的值:  " << Bing(p, q) << std::endl;
}
template <typename T>
T Bing(T a,T b){
    T c = a + b;
    return c;
}

運行結果這樣

是不是很有意思，這個函數

template <typename T>
T Bing(T a,T b){
    T c = a + b;
    return c;
}
template <typename T>
T Bing(T a,T b){
    T c = a + b;
    return c;
}

完成了兩個數據類型的操作，其實現是依靠下面這行代碼做到的:

template <typename T>

其固定格式基本如下

template <typename name>
fun_type fun_name(The value you wang to return){
    fun_body
}

很有意思的東西，衹是在OI不是很可以用到，不過很有意思啊
