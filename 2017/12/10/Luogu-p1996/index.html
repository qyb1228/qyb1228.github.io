約瑟夫問題

没有当作很难的题，不过我终究还是一只蒟蒻 ·······

所以 好難啊··· qwq

題目

题目背景

约瑟夫是一个无聊的人！！！

题目描述

n个人(n<=100)围成一圈,从第一个人开始报数,数到m的人出列,再由下一个人重新从1开始报数,数到m的人再出圈,……依次类推,直到所有的人都出圈,请输出依次出圈人的编号.

输入输出格式

输入格式:

n m

输出格式：

出圈的编号

输入输出样例

输入样例#1：

10 3

输出样例#1：

3 6 9 2 7 1 8 5 10 4

说明

m,n≤100m, n \le 100m,n≤100

總而言之，作爲一道 level1 的水題，他表現了他的價值。

思路就不説了，代碼注釋裏都有的

那就放代碼啦：

#include<iostream>
namespace bing {
    int qyb=0;
}
int main()
{
    using std::cin;
    using std::cout;
    int n, m;
    cin >> n >> m;//取值
    bool  Qyb[1001] = { 0 };//建立一個bool數組，並初始化為false
    for (int k = 0; k<n; k++) {//有n個數，要彈出n次
        for (int i = 0; i<m; i++) 
        { 
            if (++bing::qyb>n)
                bing::qyb = 1;//彈出這個數
            if (Qyb[bing::qyb])//如果這個數被彈出了
                i--;
        }
        cout <<bing::qyb << " ";
        //這個地方有兩個選擇：一 循環了m次都沒有找到bing::qyb標注為1的，那麽就直接輸出就是樣例中的3 6 9
                                          //二 有找到bing::qyb>n的數，那麽
        Qyb[bing::qyb] = true; //都輸出了，那肯定是已經把這個數去除了
    }//其實 最初都沒有找到大於n的數，但正是如此，所以 以 n 為 步數的跳。
    return 0;
}

PS: 你可以明顯的發現一個我的騷騷的操作

namespace bing{
    int qyb=0;
}

這是名稱聲明，我聲明了一個名爲 bing 的的空間，其中包含一個int類型的變量qyb

不過要是想用它，就需要這樣

bing::qyb

這樣才能實現對他的使用，就像你在開頭不使用using namespace std

然後在代碼内部用類似std::cin std::cout的操作一樣
