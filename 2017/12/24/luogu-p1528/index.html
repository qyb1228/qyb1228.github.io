切蛋糕

好难的题啊，我A的第一道0.6的题

题目

题目描述

Facer今天买了n块蛋糕，不料被信息组中球球等好吃懒做的家伙发现了，没办法，只好浪费一点来填他们的嘴巴。
他答应给每个人留一口，然后量了量每个人口的大小。Facer有把刀，可以切蛋糕，但他不能把两块蛋糕拼起来，
但是他又不会给任何人两块蛋糕。现在问你，facer怎样切蛋糕，才能满足最多的人。（facer的刀很强，切的时候
不会浪费蛋糕）。

输入输出格式

输入格式：

第一行n，facer有n个蛋糕。接下来n行，每行表示一个蛋糕的大小。再一行一个数m，为信息组的人数，然后m
行，每行一个数，为一个人嘴的大小。(1<=n<=50, 1<=m<=1024)

输出格式：

一行，facer最多可以填多少张嘴巴。

输入输出样例

输入样例#1：

4
30
40
50
25
10
15
16
17
18
19
20
21
25
24
30

输出样例#1：

7

讲解

我都不知道讲啥qwq

这题其实就是DFS+,就是要剪枝。

先把嘴的大小从小到大排序

然后用了一个小剪枝

while (bing::ans < bing::mouth_size[bing::m]) bing::m--;//剪枝

就是说，当所有蛋糕的总数bing::ans小于最后一个嘴(bing::mouth_size[bing::m])的大小时，就是m减1，会减少时间

然后二分

从前往mid搜

设了一个函数Qyb，就是DFS的Func

在Qyb里，我还设了剪枝，就是当前后的嘴的大小是相同时，我就可以剪枝，省去一次调用函数的浪费。你要注意到，我是在递归调用，所以在浪费时间的过程中，也在浪费空间啊qwq

AC程序就是这样

#include<iostream>
#include<algorithm>
#define R register
#define maxn 2500
template<typename T>
inline T Qyb(T, T);
inline void get();
inline void more();
namespace bing {
    int n, m;
    int d[maxn],qyb[maxn],mouth_size[maxn], cake_size[maxn];
    int ans;
    int wa,s, e;
    int mid;
    int p,  start;
}
int main() {
    get();
    more();
    return 0;
}
template <typename T>
inline T Qyb(T p, T start) {
    if (!p) return 1;
    if (bing::ans - bing::wa< bing::qyb[bing::mid]) return 0;
    for (int i = start; i <= bing::n; i++) {
        if (bing::d[i] >= bing::mouth_size[p]) {
            bing::d[i] -= bing::mouth_size[p];
            if (bing::d[i] < bing::mouth_size[1]) bing::wa += bing::d[i];
            if (bing::mouth_size[p] == bing::mouth_size[p - 1]) {
                if (Qyb(p - 1, i)) return 1;
            }
            else if (Qyb(p - 1, 1)) return 1;
            if (bing::d[i] < bing::mouth_size[1]) bing::wa -= bing::d[i];
            bing::d[i] += bing::mouth_size[p];
        }

    }
    return 0;
}
inline void get() {
    std::ios::sync_with_stdio(false);
    std::cin >> bing::n;
    for (int i = 1; i <= bing::n; i++) {
        std::cin >> bing::cake_size[i];//取入蛋糕大小
        bing::ans += bing::cake_size[i];//ans是蛋糕大小总和
    }
    std::cin >> bing::m;
    for (int i = 1; i <= bing::m; i++)
        std::cin >> bing::mouth_size[i];//取值
}
inline void more() {
    std::sort(bing::mouth_size + 1, bing::mouth_size + bing::m + 1);
    //排序嘴的大小
    while (bing::ans < bing::mouth_size[bing::m]) bing::m--;//剪枝
    for (int i = 1; i <= bing::m; i++) bing::qyb[i] = bing::qyb[i - 1] + bing::mouth_size[i];//qyb数组是用来取前i个人嘴的大小的
    bing::e = bing::m;
    while (bing::s <= bing::e) {
        bing::wa = 0;
        for (int i = 1; i <= bing::n; i++) bing::d[i] = bing::cake_size[i];//把蛋糕大小给d数组
        bing::mid = (bing::e - bing::s) / 2 + bing::s;
        if (Qyb(bing::mid, 1)) bing::s = bing::mid + 1;
        else bing::e = bing::mid - 1;
    }
    std::cout << bing::e;
}
